# Semantic segmentation with Dinov3

## installation

torchao requires pytorch 2.9.1.

```bash
pip install torch==2.9.1 torchvision==0.24.1 torchaudio==2.9.1 --index-url https://download.pytorch.org/whl/cu130

pip install torchao
pip install transformers
pip install datasets accelerate scikit-learn matplotlib numpy
```

## Accessing Dinov3 models

You need to submit access request to Meta thourgh Meta AI website on HuggingFace.
